# 객체 지향 프로그래밍(Object-Oriented Programming, OOP)

## 객체지향 프로그래밍이란?
클래스 단위로 프로그래밍을 하는 것.
![](../assets/OOP-1.png)
* Java 언어는 객체지향 언어이므로 객체(Object) 단위로 프로그래밍을 해야 한다. - 인간 중심 관점
* Java 언어는 객체지향 언어이므로 클래스(Class) 단위로 프로그래밍을 해야 한다. - 프로그래밍 관점

## 객체지향 설계 원칙(SOLID 원칙)
